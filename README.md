# GitHub-ready portfolio

This version converts the original single-file portfolio into a static multi-page structure suitable for GitHub Pages.

## Structure

- index.html
- styles.css
- script.js
- projects/cozymakeovers.html
- projects/weather.html
- projects/edison.html
- assets/

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In GitHub, open Settings > Pages.
4. Set the source to deploy from the main branch root.
5. Save and wait for the site URL.
